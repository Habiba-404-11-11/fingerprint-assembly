# 🔐 Fingerprint Access Control System

## 📌 Description
A **hardware-based fingerprint authentication system** using **Arduino** and a fingerprint sensor. The system verifies user identity and provides visual feedback using LEDs to indicate access status.

---

## ⚙️ Features
- Fingerprint-based authentication  
- Access granted (Green LED)  
- Access denied (Red LED)  
- Real-time hardware interaction  
- Secure identity verification  

---

## 🧠 Technologies & Components
- Arduino (Microcontroller)  
- Fingerprint Sensor Module  
- LEDs (Green & Red)  
- Breadboard & Connecting Wires  
- Embedded Programming  

---

---

## 🚀 How It Works
1. User places finger on the fingerprint sensor  
2. Arduino processes the fingerprint data  
3. If the fingerprint matches:
   - ✅ Green LED turns ON (Access Granted)  
4. If not:
   - ❌ Red LED turns ON (Access Denied)
